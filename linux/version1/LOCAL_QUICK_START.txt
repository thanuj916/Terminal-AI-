STEP 1: Install Ollama (5 minutes)

  ollama pull orca-mini     # 1.7 GB - Good for weak systems

  ollama pull llama2        # 3.8 GB - Solid choice

QUICK REFERENCE

  help                   Show all commands
  history                View command history
  config                 Show configuration
  models                 List installed models
  backend                Show which backend
  ? <question>           Ask a question
  !!                     Repeat last command
  exit                   Exit program

════════════════════════════════════════════════════════════════════════════

TROUBLESHOOTING QUICK FIXES

"Cannot connect to Ollama"
  → Make sure Ollama is running: ollama serve

"Model not found"
  → Pull it again: ollama pull gemma2

"Too slow"
  → Try faster model: ollama pull mistral
  → Or: ollama pull orca-mini

"Permission denied"
  → Fix with: chmod +x ~/.local/bin/ai-terminal

════════════════════════════════════════════════════════════════════════════

MODEL SWITCHING (Advanced)

Want to try a different model?

1. Pull it:
   ollama pull mistral

2. Edit config:
   nano ~/.ai_terminal_config
   
3. Change model line:
   "model": "mistral"
   
4. Restart AI terminal:
   Exit and rerun ai-terminal
